## FULL FEATURE LIST

### Voice AI Receptionist (core)
- Answers every call 24/7 in a natural, customizable voice
- Collects customer name, phone, pet name, breed, size, service, preferred time
- Checks real-time calendar availability during the call
- Quotes pricing instantly using dynamic breed x size x service matrix
- Books the appointment directly into the calendar
- Reads back confirmation and sends SMS automatically
- Recognizes returning customers by phone — "Hi Sarah, booking Max again?"
- Configurable personality: warm / professional / playful
- Multi-language: English, Spanish, French, and more
- Custom greeting and fallback messages

### Smart Booking Engine
- **Hard mode:** Instant confirmed booking + calendar event + SMS confirmation
- **Soft mode:** Tentative hold with one-tap SMS confirmation link (2-hour expiry)
- Captures: pet details, service, price, groomer, start/end time

### Calendar Integrations
- **Google Calendar** (OAuth) — primary for most shops
- **Square Appointments** (OAuth) — for Square POS users
- **Acuity Scheduling** (OAuth) — for Acuity users
- **Gingr / MoeGo compatible** — connect via shared Google Calendar
- Conflict checking with buffer time (15-min optional)
- Respects "busy" events on personal calendars

### SMS Command System (owner + customer)
- Customers text: `STATUS`, `CONFIRM`, `CANCEL`, `REBOOK`, `BOOK`
- Owner texts: `CANCEL [phone]`, `CONFIRM [phone]`
- "Is My Dog Ready?" — live grooming status updates via text

### No-Show Protection (4 layers)
- 48-hour SMS reminder with one-tap confirm/cancel
- 24-hour follow-up reminder
- No-response escalation to owner
- One-click no-show marking on dashboard
- Repeat offender tracking (2+ no-shows flagged)
- 30-day stats: no-show rate, confirmed count, revenue saved

### Waitlist Auto-Fill
- Cancellation triggers automatic waitlist notification
- First-come-first-served SMS: "A spot opened up — reply BOOK to grab it"
- Owner notified of backfill in real time

### New Client Intake Forms
- Auto-sent via SMS to first-time customers after booking
- Collects: pet details, vaccines, vet info, allergies, bite history, emergency contact
- No login required — public link with unique token

### Smart Rebooking Engine
- Breed-aware reminders (default 6-week cycle, fully configurable)
- Auto-SMS when a dog is due back and no future appointment exists
- "Reply BOOK" one-tap rebooking

### Lapsing Client Re-engagement
- Auto-identifies clients overdue for their next groom
- One-click "Text All" blast to every lapsing client
- Color-coded urgency (amber = overdue, red = 90+ days)

### Google Review Automation
- Auto-requests reviews 2 hours after appointment completion
- Trackable redirect links with click analytics
- Paste your Google Business URL — system handles the rest

### Missed Call Auto-Reply
- If a call ends without a booking: caller gets an SMS recovery text
- Owner gets notified with caller info
- Turns every missed opportunity into a second chance

### Customer Memory
- AI remembers: name, pet, breed, preferred service, visit count, VIP status
- Behavioral notes (anxious, muzzle required, special handling)
- No-show history surfaced to AI during calls

### Dynamic Pricing Matrix
- Set base prices per service
- Override by breed + size + service (e.g., Golden Retriever Large Full Groom = $95)
- AI quotes the correct price live on every call

### Dashboard & Analytics
- Calls this week/month, confirmed bookings, missed calls
- Average call duration, revenue protected
- Today's view with live grooming status per appointment
- Full call log with transcript search, AI summaries, and recording playback

### Behavioral Risk Logging
- Severity levels: Note / Caution / High Risk
- Tags: muzzle_required, anxious, biting, reactive, special_handling
- AI reads behavior brief before each call

### Agent Personality Settings
- Voice selection, greeting, fallback message
- Tone (warm / professional / playful), style (concise / detailed)
- Language selection, custom system prompt override
